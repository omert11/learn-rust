# Rust Öğrenme Projeleri

### Genel Bakış
Bu repo, Rust programlama dilini öğrenmek için geliştirilen çeşitli projeleri içermektedir. Her proje, Rust'ın farklı özelliklerini ve kavramlarını öğrenmeyi amaçlamaktadır.


### Projeyi Çalıştırma
```bash
cargo run
```

### Test Etme
```bash
cargo test
```
